## What is a Unit?

A **Unit** (also called a course folder or course group) is an organizational container within a course. It groups related assignments, readings, and other content together - typically by week, module, topic, or concept.

## Unit Information

### Tooltip Information

When you hover over a unit, you'll see detailed information:

```
Unit: Week 1 - Introduction
5 items
Type: Lab
2 unread messages
```

**What each line means:**

1. **Unit: [Name]** - The unit title
2. **[X] items** - How many assignments, readings, or sub-units are inside
3. **Type: [Content Type]** - The category (Lab, Homework, Exam, etc.) set by the instructor
4. **[X] unread messages** - Number of new announcements or discussions (only shown if > 0)

### Description in Tree

## Available Actions

Right-click on a unit to access:

### Show Messages
View announcements and discussions specific to this unit. Instructors may post unit-specific information, clarifications, or updates here.

**When to use:**
- Check for unit-specific announcements
- Ask questions about the unit content
- View instructor clarifications

## Content Types and Colors

Different units may have different colored icons based on their **content type**:

- Instructors define content types (e.g., "Homework", "Lab", "Exam", "Project")
- Each type gets a unique color
- This visual coding helps you quickly identify the type of work

## Messages and Notifications

### Unread Message Indicator

When a unit has unread messages, you'll see:

```
âš« Week 1    ğŸ”” 2 â€¢ 5 items
```

The **ğŸ”” bell icon** with a number tells you how many messages you haven't read yet.

**What messages might include:**
- Unit-specific announcements
- Clarifications about assignments in this unit
- Deadline extensions
- Tips and hints
- Instructor feedback

### Checking Messages

1. Right-click on the unit
2. Select **Show Messages**
3. Read and respond to messages
4. The unread count will update automatically

## Tips

- **Expand units to see contents** - Click the arrow to reveal assignments and readings
- **Check unread messages promptly** - Important information might be time-sensitive
- **Units organize your work** - Think of them as weekly modules or topic sections
- **Nested structure is intentional** - Instructors organize content logically to guide your learning
